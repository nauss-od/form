# منصة تأمين المشاركين للدورات الخارجية

نسخة معاد تنظيمها معماريًا فوق الواجهة المعتمدة، مع قاعدة بيانات Prisma/PostgreSQL ونقاط التوسع اللازمة للمرفقات والتصدير.

## ملاحظات مهمة
- هذه الحزمة تضع الهيكل المعماري النهائي وتوحّد الحقول والمنطق المطلوب.
- ما زالت بعض وظائف التصدير والتخزين الخارجي للمرفقات موضوعة كـ placeholders آمنة لتكتمل لاحقًا بدون كسر المشروع.
- يمكن استبدال ملفات مشروع `forms` الحالي بهذه النسخة داخل نفس المستودع.

## التشغيل
```bash
npm install
npx prisma db push
npm run build
npm run dev
```
